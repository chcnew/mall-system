# 4.3 后台操作手册

## 二、商品管理

### 2.1 添加商品

**步骤**:
1. 点击"商品管理" → "商品列表"
2. 点击"添加商品"按钮
3. 填写商品信息:
   - 商品名称（必填）
   - 商品副标题
   - 商品分类（必填）
   - 商品价格（必填）
   - 原价
   - 商品图片（至少上传一张）
   - 商品规格
   - 商品库存
   - 商品描述
4. 点击"保存"按钮

### 2.2 编辑商品

**步骤**:
1. 进入商品列表
2. 找到要编辑的商品
3. 点击"编辑"按钮
4. 修改商品信息
5. 点击"保存"按钮

### 2.3 上架/下架商品

**步骤**:
1. 进入商品列表
2. 勾选要操作的商品
3. 点击"上架"或"下架"按钮

---

## 三、订单管理

### 3.1 查看订单

**步骤**:
1. 点击"订单管理" → "订单列表"
2. 筛选订单（可选）
3. 查看订单信息

### 3.2 订单发货

**步骤**:
1. 进入订单列表
2. 找到"待发货"状态的订单
3. 点击"详情"按钮
4. 点击"发货"按钮
5. 选择物流公司
6. 输入物流单号
7. 点击"确认发货"

### 3.3 订单退款

**步骤**:
1. 进入订单列表
2. 找到要退款的订单
3. 点击"详情"按钮
4. 点击"退款"按钮
5. 填写退款原因
6. 点击"确认退款"

---

## 四、用户管理

### 4.1 查看用户列表

**步骤**:
1. 点击"用户管理" → "用户列表"

## 五、营销管理

### 5.1 创建优惠券

**步骤**:
1. 点击"营销管理" → "优惠券管理"
2. 点击"添加优惠券"
3. 填写优惠券信息:
   - 优惠券名称
   - 优惠券类型
   - 优惠值
   - 最低消费
   - 发放总量
   - 每人限领
   - 有效期
4. 点击"保存"

## 七、常见问题

### Q: 商品图片无法上传？
A: 检查图片格式（支持 jpg、png、gif）和大小（不超过 5MB）

### Q: 订单发货后物流信息不更新？
A: 系统会自动查询物流信息，请稍后刷新查看

### Q: 如何重置管理员密码？
A: 联系系统管理员进行密码重置

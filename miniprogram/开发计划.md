## 第一周：核心页面开发

### Day 1-2：商品详情页（最重要）

#### 页面结构
```
pages/product/
├── index.js
├── index.wxml
├── index.wxss
└── index.json
```

#### 核心功能
- 商品信息展示（主图、价格、销量等）
- SKU 规格选择器
- 数量选择器
- 添加购物车功能
- 收藏功能
- 图片轮播展示
- 商品参数说明
- 相关推荐商品

#### 交互流程
1. 进入页面 → 加载商品数据
2. 选择 SKU 规格 → 更新价格和库存
3. 选择数量 → 检查库存
4. 添加到购物车 → 成功提示
5. 立即购买 → 跳转到订单确认页

#### 预览效果
- 完整的商品展示界面
- 流畅的规格选择体验
- 清晰的价格变化反馈

##  开发技巧

### 快速看到效果的建议

1. **使用模拟数据**
   - 在每个页面的 `onLoad` 方法中使用 `setTimeout` 加载模拟数据
   - 使用占位图片展示界面效果

2. **关键交互优先开发**
   - 核心流程优先实现
   - 异常情况提示完善
   - 加载状态显示

3. **页面跳转优先完善**
   - 确保页面间跳转路径正确
   - 传递必要的参数

4. **模拟数据模板**
```javascript
const mockData = {
  id: 1,
  name: '测试商品',
  price: 99.00,
  stock: 100,
  images: [],
  description: '商品描述'
}
```

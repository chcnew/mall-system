# 微信小程序商城 - 立即运行

## 🚀 快速开始（3步）

### 1. 打开微信开发者工具
下载地址：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

### 2. 导入项目
- 点击 "+" 或 "导入项目"
- 选择目录：`D:\CodeFile\mall-system\miniprogram`
- AppID：选择"测试号"（无需申请）

### 3. 编译运行
点击"编译"按钮，直接运行！

## 🔧 问题排查

### 页面显示空白？
1. 确保选择"测试号"
2. 点击"清缓存" → "清除全部缓存"
3. 重新编译

### 编译报错？
查看控制台（Console）输出，确认具体错误信息

### 图片不显示？
这是正常的，我们使用彩色占位符，不影响功能使用

## 📝 技术栈

- **架构**: 模块化、组件化
- **数据**: 模拟数据（setTimeout）

---

## 🎯 数据说明

当前所有接口使用 **setTimeout 模拟后端响应**，数据为静态 Mock。

如需对接真实后端：
1. 修改 `utils/request.js` 中的 `BASE_URL`
2. 取消注释 API 请求代码
3. 启动 FastAPI 后端服务

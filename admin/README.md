# 商城后台管理系统

基于 Vue 3 + TypeScript + Vite + Element Plus 开发的商城后台管理系统。

## 技术栈

- Vue 3 (Composition API)
- TypeScript
- Vite
- Element Plus
- Pinia
- Vue Router
- Axios
- ECharts
- @vueuse/core

## 安装依赖

```bash
npm install
```

## 开发

```bash
npm run dev
```

访问 http://localhost:5173

## 构建

```bash
npm run build
```

## 默认账号

- 用户名: admin
- 密码: admin123

- 控制台

  - 用户详情
